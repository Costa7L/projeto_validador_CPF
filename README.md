# Validador de CPF

## Sobre o projeto

Este projeto foi desenvolvido em Python com o objetivo de validar números de CPF utilizando o algoritmo oficial de verificação dos dígitos verificadores.

O programa recebe um CPF informado pelo usuário, remove caracteres de formatação (pontos e traços) e espaços indesejados, verifica se o CPF possui 11 dígitos válidos e realiza o cálculo dos dígitos verificadores para determinar se o documento é válido ou inválido.

## Funcionalidades

* Receber CPF digitado pelo usuário.
* Remover pontos, traços e espaços automaticamente.
* Verificar se o CPF possui 11 dígitos.
* Verificar se contém apenas números.
* Rejeitar CPFs com todos os dígitos iguais.
* Validar os dois dígitos verificadores utilizando a regra oficial do CPF.
* Exibir o resultado da validação.
